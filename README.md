# 🏢 Sub-Category Reduction at Superstore

## 📚 About the Data
Superstore, a global fictional retailer, has been concerned about the growth of product category (and sub-categories) that they offer. In an attempt to keep it under control, they provided their sales data from 2012 - 2015 with the task of removing 2-3 sub-categories from their offerings.
Each row in the data represents an individual sale for a specific product (with 1+ units ordered).

## 💡 Highlights
Sub-categories were evaluated on three criteria:
- Total number of orders
- Average discount rate
- Profit-to-sales ratio
Two sub-categories consistently stood out negatively:
- **Tables**
  - Only unprofitable sub-category
  - Highest discount rate (29.1%)
  - Lowest number of orders (861)
- **Machines**
  - Second lowest profit-to-sales ratio (8%)
  - Above average discount rate (17%)
  - Second lowest number of orders (1,486)
